# 茄子学习系统 v2.0｜Ability Framework

## 设计目标

从 v2.0 起，长期能力追踪逐步从百分制转向 L1~L10 能力等级。

百分制适合一次诊断，L 等级适合长期追踪。一个能力项可以因为稳定表现上升，也可以因为遗忘、迁移失败或连续错误下降。

## L1~L10 定义

- L1：完全陌生，需要从零讲解。
- L2：听过概念，但不能独立识别。
- L3：能在提示下完成基础题。
- L4：能独立完成标准基础题，但不稳定。
- L5：能稳定完成基础题，开始处理简单变式。
- L6：能完成常规变式题，知道常见易错点。
- L7：能进行跨知识点迁移，表达较完整。
- L8：能处理重点高中常规综合题，速度和准确率较稳定。
- L9：能处理较高难度综合题，并能解释方法选择。
- L10：能稳定解决高难题，具备反思、归纳和举一反三能力。

## 等级更新规则

能力等级不能只凭单题表现更新，应综合：

- 最近 3 次相关练习表现
- 是否独立完成
- 是否需要提示
- 是否能讲出理由
- 是否能迁移到变式
- 是否能在限时条件下保持稳定

如果信息不足，等级标注为“待评估”。
