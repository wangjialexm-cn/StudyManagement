## 2026-06-28｜Day5 英语教学复盘

### 最大发现

不是所有英语错误都是语法错误。

Day4 中 `be increase / be decrease` 表面上像 `be doing` 结构错误，但 Day5 批改后确认，学生已经能正确完成：

- `The number is increasing.`
- `The price is decreasing.`

因此，真正根因不是不会 `be doing`，而是：

- 学生认识 `increase / decrease` 的中文意思；
- 但不知道 `increase / decrease` 本身就是动词；
- 词汇没有进入主动输出阶段。

### 教学判断

- `be doing`：基础掌握（Basic Mastery）。
- 词汇主动输出：当前重点突破（Priority）。

### 后续调整

从 Day6 开始，英语训练重点转向 Lexical Radar：

1. 词性识别；
2. 动词形式变化；
3. 固定搭配；
4. 主动造句。

降低 `be doing` 重复练习比例，增加词汇深度训练比例。

### 教研原则

后续批改英语题时，不能只看表面错误类型。

应先判断：

1. 学生是否认识单词；
2. 学生是否知道词性；
3. 学生是否知道常见搭配；
4. 学生是否能把词放进句子主动输出。

若学生“认识中文意思但不会入句”，优先按词汇主动输出问题处理，而不是直接归为语法错误。
