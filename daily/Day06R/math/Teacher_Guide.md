# Day6R 数学 Teacher Guide｜函数第一课

## 本节定位

按重点高中实验班节奏，直接建立函数第一课完整框架。集合复习只作为定义域和值域的入口，不占主线。

## 教学目标

1. 建立函数定义和唯一性意识。
2. 一次讲完整函数三要素。
3. 训练定义域和值域基本题。
4. 引入三种表示法和函数判断。

## 时间建议（70-85分钟）

- 0-5 min：集合到函数的过渡。
- 5-20 min：函数定义和唯一性。
- 20-35 min：三要素。
- 35-50 min：定义域和值域。
- 50-60 min：三种表示法。
- 60-75 min：判断函数和综合题。
- 75-85 min：总结与答疑。

## 课堂板书

函数第一课

1. 定义：x 唯一对应 y
2. 三要素：定义域、对应关系、值域
3. 定义域：分母不为0，根号内非负
4. 值域：y 的范围
5. 表示法：解析式、表格、图像
6. 判断函数：唯一性 / 竖线检验

## 提问设计

- 为什么“唯一”是函数定义的核心？
- 两个函数解析式一样就一定相同吗？
- 求定义域第一步看什么？
- 图像怎样判断是不是函数？

## 易错提醒

- 学生可能用初中经验只看公式，不看定义域。
- 值域容易直接照抄定义域。
- 判断函数时容易忽略同一个 x 对应多个 y。
