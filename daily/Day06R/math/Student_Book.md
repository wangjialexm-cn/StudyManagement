## 为什么学习

集合之后，高中数学真正进入主线：函数。初中学过一次函数、二次函数，但高中函数不只是画图和代公式，而是研究“一个量如何由另一个量决定”。

函数是高中数学的核心语言。后面的指数函数、对数函数、三角函数、导数、数列、解析几何，都会用函数思想。

## 学习目标

1. 理解函数的定义：每一个自变量对应唯一函数值。
2. 掌握函数三要素：定义域、对应关系、值域。
3. 会求简单函数定义域和值域。
4. 会读懂解析式、表格、图像三种表示法。
5. 会判断一个对应关系是不是函数。
6. 能做函数第一课综合题。

## 自动复习｜5分钟

集合语言在函数中继续使用：

- 定义域是自变量 x 能取的集合。
- 值域是函数值 y 能取的集合。
- 判断范围时仍要有条件检索意识。

## 知识块一：函数是什么

一般地，设 A、B 是两个非空数集，如果按照某种确定的对应关系 f，使集合 A 中任意一个数 x，在集合 B 中都有唯一确定的数 y 和它对应，那么称 f 是从 A 到 B 的一个函数。

记作：

`y=f(x), x 属于 A`

关键词：

- 任意一个 x；
- 唯一确定的 y；
- x 的取值范围是定义域。

高考教练告诉你：判断函数不是看公式像不像，而是看“一个 x 是否只对应一个 y”。

## 知识块二：函数三要素

函数三要素：

1. 定义域：x 允许取哪些值。
2. 对应关系：x 如何变成 y。
3. 值域：y 最终能取哪些值。

两个函数相同，必须三要素都相同。只看解析式不够。

例：

- f(x)=x+1，x 属于 R；
- g(x)=x+1，x 属于 {1,2,3}。

它们对应关系相同，但定义域不同，所以不是同一个函数。

## 知识块三：定义域

求定义域先看限制：

1. 分母不能为 0。
2. 偶次根号内不能小于 0。
3. 实际问题要符合题意。

例：`f(x)=1/(x-2)`，定义域是 `x 不等于 2`。

例：`f(x)=sqrt(x-1)`，定义域是 `x>=1`。

方法总结：定义域不是随便写 R，要先检查限制条件。

## 知识块四：值域

值域是 y 的取值范围。

例：f(x)=x+1，x 属于 {1,2,3}，则 y 属于 {2,3,4}。

例：f(x)=x^2，x 属于 R，则 y>=0。

值域比定义域更难，因为它要看函数值能到哪里。

## 知识块五：三种表示法

函数可以用三种方式表示：

1. 解析式法：y=2x+1。
2. 列表法：列出 x 和 y 的对应表。
3. 图像法：用平面直角坐标系表示。

三种表示法本质相同：都在描述 x 和 y 的对应关系。

## 知识块六：判断是不是函数

判断标准：

一个 x 只能对应一个 y。

是函数：

- x=1 对应 y=2；
- x=2 对应 y=4；
- x=3 对应 y=6。

不是函数：

- x=1 同时对应 y=2 和 y=3。

图像判断：任意一条竖直线与图像最多有一个交点，则是函数图像。

## 例题精讲

### 例1｜判断函数

对应关系：x=1 对应 y=2；x=2 对应 y=3；x=1 对应 y=4。它是不是函数？

解：不是。因为同一个 x=1 对应了两个不同 y。

### 例2｜求定义域

求 `f(x)=1/(x+3)` 的定义域。

解：分母 x+3 不能为 0，所以 x 不等于 -3。

### 例3｜求值域

f(x)=x^2，x 属于 { -1,0,2 }，求值域。

解：当 x=-1 时 y=1；x=0 时 y=0；x=2 时 y=4。值域为 {0,1,4}。

## 高考风格题

已知 f(x)=1/(x-a)，若函数定义域不能取 x=3，则 a 的值是多少？

思路：分母为 0 的点被排除，所以 x-a=0 的解是 3。

## 方法总结

函数第一课四步：

1. 看定义域；
2. 看对应关系；
3. 看值域；
4. 判断一个 x 是否唯一对应一个 y。

## 最容易错的地方

- 把对应关系相同误认为函数相同。
- 求定义域忘记分母不能为 0。
- 求值域只看 x，不计算 y。
- 判断函数时忽略“唯一”。
- 图像题忘记竖线检验法。

函数

├─ 表示法：解析式、列表、图像

## 本节一句话总结

函数的核心不是公式，而是“定义域中的每一个 x 都唯一确定一个 y”。
