# Day6R 数学 Class Practice｜函数第一课

## A组｜函数概念

1. 判断：x=1 对应 y=2，x=2 对应 y=4，x=3 对应 y=6，是否为函数？
2. 判断：x=1 对应 y=2，x=1 对应 y=3，是否为函数？
3. 函数定义中“唯一确定”是什么意思？
4. 为什么函数必须先说明 x 的取值范围？

## B组｜三要素

5. 写出函数三要素。
6. f(x)=x+2，x 属于 R；g(x)=x+2，x 属于 {1,2}。f 和 g 是否为同一函数？
7. f(x)=2x，定义域 {1,2,3}，求对应值。
8. 函数相同必须满足什么条件？

## C组｜定义域

9. 求 f(x)=1/(x-1) 的定义域。
10. 求 f(x)=1/(x+5) 的定义域。
11. 求 f(x)=sqrt(x) 的定义域。
12. 求 f(x)=sqrt(x-2) 的定义域。

## D组｜值域

13. f(x)=x+1，x 属于 {1,2,3}，求值域。
14. f(x)=x^2，x 属于 {-2,0,1}，求值域。
15. f(x)=2x-1，x 属于 {0,1,2}，求值域。

## E组｜综合题

16. 判断关系表是否为函数：{(1,2),(2,3),(3,3)}。
17. 判断关系表是否为函数：{(1,2),(1,4),(2,5)}。
18. 已知 f(x)=1/(x-a)，若 x=4 不在定义域内，求 a。
19. 已知 f(x)=x^2+1，x 属于 {-1,0,1,2}，求值域。
20. 用一句话说明：函数和集合有什么关系？
