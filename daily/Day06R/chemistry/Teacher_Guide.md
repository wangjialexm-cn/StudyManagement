# Day6R 化学 Teacher Guide｜物质的量基础

## 本节定位

按重点高中实验班节奏，一次建立物质的量基础框架。不要把 mol、M、NA、N 拆得过碎。

## 教学目标

1. 讲清 mol 的计数意义。
2. 建立 M、NA、N、n 的关系。
3. 训练公式链 m -> n -> N。
4. 强化化学式下标理解。

## 时间建议（45-55分钟）

- 0-5 min：CO2 摩尔质量快速复习。
- 5-15 min：mol 与 NA。
- 15-25 min：摩尔质量 M。
- 25-40 min：N=nNA 与 n=m/M。
- 40-55 min：综合计算。

## 板书

物质的量基础

1. 1 mol = 6.02x10^23 个粒子
2. M：g/mol
3. n=m/M
4. N=nNA
5. CO2：1C + 2O
