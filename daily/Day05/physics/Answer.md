# Day5 物理 Answer

## Class Practice

1. 平均速度等于某段运动中总路程与总时间的比值。解析：表达题必须出现“总路程、总时间、整个过程”的意思。
2. 因为平均速度描述全程，要看总路程和总时间，不是简单平均速度数值。解析：若两段用时不同，直接平均速度会错。
3. v=100/20=5 m/s。解析：代入 v=s/t。
4. v=300/60=5 m/s。解析：总路程300 m，总时间60 s。
5. 总路程100 m，总时间40 s，v=2.5 m/s。解析：先加路程和时间，再相除。
6. 总路程150 m，总时间50 s，v=3 m/s。解析：分段运动先合并总量。
7. v=50/10=5 m/s。解析：图像中路程变化量为50 m，时间变化量为10 s。
8. 路程变化30 m，时间变化10 s，v=3 m/s。解析：不能只看终点，要看变化量。
9. 路程变化是 80 m。解析：图像中 0 秒路程为 0 m，8 秒路程为 80 m。
10. 平均速度 v=80/8=10 m/s。解析：用路程变化量除以时间变化量。
11. 总路程120 m，总时间20 s，v=120/20=6 m/s。解析：分段运动仍然先合并总路程和总时间。

## Homework

1. 平均速度描述整个运动过程，等于总路程除以总时间。解析：这是 Day5 物理最重要的表达模板。
2. v=120/30=4 m/s。解析：直接代入公式。
3. v=250/50=5 m/s。解析：单位为 m/s。
4. 错。解析：平均速度应看总路程和总时间，不一定等于速度平均值。
5. 总路程200 m，总时间60 s，v=200/60=10/3 m/s。解析：分段题先合并。
6. 总路程150 m，总时间40 s，v=150/40=3.75 m/s。不能直接平均两个速度，因为两段时间不同。
7. v=48/12=4 m/s。解析：图像中用路程变化量除以时间变化量。
8. 路程变化60 m，时间变化15 s，v=4 m/s。解析：90-30=60，20-5=15。
9. 小明的问题是忽略了每段所用时间。解析：平均速度描述全程，应为总路程除以总时间，而不是速度数值的简单平均。
10. v=120/20=6 m/s。解析：图像中 0 到 20 秒路程从 0 m 增加到 120 m。
11. 总路程300 m，总时间60 s，v=300/60=5 m/s。不可以直接平均两段速度，因为平均速度看全程总路程和总时间。
