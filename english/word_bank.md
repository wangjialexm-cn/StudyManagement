# 英语重点词汇库｜Word Bank

## 文件用途

本文件统一整理茄子的英语重点词汇，合并旧词汇表、Codex 学习记录、课堂真实错题和后续复现需求。

记录原则：

- 同一个词只保留一个主条目，历史问题写入“来源/证据”。
- “掌握状态”只根据已有证据判断，不把教材出现过的词直接记为已掌握。
- 真实错题优先级高于普通词汇复习。
- Day6 以后英语教材每天应从本文件抽取旧词复现。

## 掌握状态说明

| 状态 | 含义 |
|---|---|
| 当前重点复现 | 近期真实课堂或批改暴露问题，必须连续复现 |
| 优先回炉 | 旧资料中反复出错，需要重新巩固 |
| 拼写不稳 | 中文到英文主动拼写不稳定 |
| 低频抽查 | 近期相对稳定，但不能视为永久掌握 |
| 待输出验证 | 已学习或会认，但缺少主动造句证据 |
| 信息不足 | 现有资料不足以判断 |
