# 英语复习计划｜Review Plan

## 文件用途

本文件根据 `english/lexical_radar.md`、`english/word_bank.md` 和 `english/error_bank.md` 生成英语后续复习安排。

核心目标：

- 让重点词真正进入句子系统；
- 让真实错题循环复现；
- 让英语课从“背单词”升级为“词汇 + 语法 + 阅读 + 翻译 + 写作”的综合输出训练。

## 当前优先级

1. `increase / decrease`：词性判断 + 动词身份确认 + 主动造句。
2. `depend on`：固定搭配 + 第三人称单数输出。
3. `discover / especially`：会认词进入造句系统。
4. 动词变形：在确认词性后训练原形、三单、现在分词、过去式等形式。
5. 旧 A 级词：每天少量回炉，不一次性堆太多。
6. 易混词：每次 2～3 组，放入句子中辨析。

> Day5 批改后更新：`be doing` 已基本掌握，Day6 起降低现在进行时重复练习比例，增加词汇深度训练比例。

## 每日输出训练

### 固定训练路径

1. 看词判断词性。
2. 写中文意思。
3. 写一个常见搭配。
4. 仿写一句。
5. 自主造一句。
6. 用 3～5 个词写一个小段。

### Day6 建议输出任务

使用以下词写 5 句：

- increase
- decrease
- depend on
- discover
- especially

要求：

- 每句先标出关键词词性；
- 至少 2 句体现动词形式变化，如三单、现在分词或过去式；
- 至少 1 句使用第三人称单数；
- 至少 1 句使用 `especially` 放在句中；
- 句子必须完整，有主语和谓语。
