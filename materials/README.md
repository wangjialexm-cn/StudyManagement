# materials 目录说明

本目录保存正式教材、课堂练习、课后训练卷和教师答案。

## 当前结构

```text
materials/
└── bridge/
    ├── day01/
    ├── day02/
    └── dayXX/
```

## 每日材料固定包含

- 学生教材
- 课堂练习
- 课后训练卷
- 教师答案

## 维护规则

1. 所有材料必须基于 `system/Material_Standard.md`。
2. 所有 Word 材料必须优先使用 `templates/` 中的模板。
3. “已生成材料”不等于“学生已完成”。
4. 学生完成、批改、总结后，才同步进入学习档案。
