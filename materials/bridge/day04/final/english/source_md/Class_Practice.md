# Day4 英语 Class Practice

## A组｜depend on

1. 改错：`Success depend on practice.`

2. 中译英：成功取决于努力。

3. 中译英：结果取决于练习。

## B组｜increase / decrease

4. 中译英：这个数字增加了。

5. 中译英：价格下降了。

6. 用 `increase` 写一句话。

## C组｜discover / especially

7. 中译英：我发现了一个好方法。

8. 中译英：我尤其喜欢这个例子。

9. 用 `especially` 写一句话。

## D组｜主动输出检查

10. 写一句包含 `depends on` 的句子。

11. 写一句包含第三人称单数动词的句子。
