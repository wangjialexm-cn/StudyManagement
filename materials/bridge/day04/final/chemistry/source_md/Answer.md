# Day4 化学 Answer

## Class Practice 答案

1. `H₂O = 2×1+16=18 g/mol`。
2. `CO₂ = 12+2×16=44 g/mol`。不是 40，因为有两个氧原子。
3. `O₂ = 2×16=32 g/mol`。
4. `H₂ = 2×1=2 g/mol`。
5. `SO₂ = 32+2×16=64 g/mol`。
6. `CaCO₃ = 40+12+3×16=100 g/mol`。
7. 因为下标表示该元素原子的个数，计算质量时必须乘进去。
8. 找元素 → 看下标 → 先乘 → 后加 → 写单位。

## Homework 答案

1. `H₂ = 2 g/mol`。
2. `O₂ = 32 g/mol`。
3. `H₂O = 18 g/mol`。
4. `CO₂ = 44 g/mol`。
5. `NaCl = 23+35.5=58.5 g/mol`。
6. `SO₂ = 32+2×16=64 g/mol`。
7. `CaCO₃ = 40+12+3×16=100 g/mol`。
8. `H₂SO₄ = 2×1+32+4×16=98 g/mol`。
9. CO₂ 中有两个氧原子，应为 `12+2×16=44`，不是 `12+16`。
10. 因为摩尔质量连接质量和物质的量，是后续 `n=m/M` 的基础。
