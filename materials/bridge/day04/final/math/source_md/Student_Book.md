# Day4 数学 Student Book｜集合（二）

## 课程定位

本课是集合章节第二课。Day3 已完成集合概念、三大特性、列举法、描述法、空集。Day4 不再慢速重复概念，而是进入集合之间的关系和高考常见题型。

建议课堂时长：60～80 分钟。

## 学习目标

学完本课，茄子应能：

1. 区分 `∈`、`⊆`、`⊂`。
2. 理解子集、真子集、相等集合。
3. 掌握空集的重要性质。
4. 会用区间表示简单数集。
5. 会用数轴法判断集合包含关系。
6. 能处理高考集合第一类题：集合关系与区间包含。

## 为什么学习这个知识点

Day3 解决的是“集合里有什么”。Day4 解决的是“集合和集合之间是什么关系”。

后面学习函数定义域、不等式解集、参数范围时，经常需要判断：

- 一个范围是否包含另一个范围；
- 两个集合是否相等；
- 某个区间是否满足题目条件。

这些都离不开子集、真子集、相等集合和数轴法。

## 高考教练告诉你

集合题常出现在高考选择题前几题。它不靠复杂计算拉开差距，而靠概念精确度拉开差距。

常见考法：

1. 给两个集合，判断关系。
2. 一个集合用描述法，一个集合用列举法，判断是否相等。
3. 给两个不等式集合，用数轴判断包含关系。
4. 写子集或真子集时考空集。

高考集合题最怕“差不多懂”。符号、端点、空集，一错就丢分。

## 知识块一：符号升级

Day3 用到 `∈`，它表示“元素属于集合”。

Day4 要新增两个符号：

- `A ⊆ B`：A 是 B 的子集。
- `A ⊂ B`：A 是 B 的真子集。

### 方法总结

先看符号两边是什么：

- 左边是一个元素：用 `∈` 或 `∉`。
- 左边是一个集合：用 `⊆`、`⊂` 或 `=`。

例：

- `2 ∈ {1,2,3}`
- `{2} ⊆ {1,2,3}`

### 最容易错的地方

把 `2 ∈ A` 写成 `2 ⊆ A`，或者把 `{2} ⊆ A` 写成 `{2} ∈ A`。

### 知识联系

符号区分以后会继续出现在函数定义域、值域、解集表达中。看到 `x∈A` 时，要知道这是“元素与集合”的关系；看到 `A⊆B` 时，要知道这是“集合与集合”的关系。

## 知识块二：子集

如果集合 A 中的每一个元素都属于集合 B，就说 A 是 B 的子集，记作：

`A ⊆ B`

例：

`A={1,2}`，`B={1,2,3}`，则 `A ⊆ B`。

### 方法总结

判断 `A ⊆ B`：

1. 只检查 A 的元素；
2. 看 A 的每一个元素是否都在 B 中；
3. 全都在，才成立。

### 最容易错的地方

只看到 A 和 B 有共同元素，就以为 A 是 B 的子集。  
子集要求“每一个元素都在”，不是“有交集”。

### 知识联系

子集以后会继续出现在函数定义域包含、参数范围判断、概率事件包含、集合综合题中。可以记成一条线：

`子集 → 函数定义域 → 参数范围 → 概率事件 → 集合综合题`

## 知识块三：真子集

如果 `A ⊆ B`，并且 `A ≠ B`，就说 A 是 B 的真子集，记作：

`A ⊂ B`

例：

`{1,2} ⊂ {1,2,3}`

但：

`{1,2,3} ⊆ {1,2,3}` 成立，`{1,2,3} ⊂ {1,2,3}` 不成立。

### 方法总结

判断真子集分两步：

1. 先判断是不是子集；
2. 再判断两个集合是否不相等。

### 最容易错的地方

把“子集”直接当成“真子集”。子集允许相等，真子集不允许相等。

### 知识联系

真子集以后常用于判断“范围更小但不完全相同”。例如参数范围题中，若一个条件范围必须严格落在另一个范围里，就要想到真子集。

## 知识块四：相等集合

如果两个集合元素完全相同，就相等。

也可以说：

如果 `A ⊆ B` 且 `B ⊆ A`，那么 `A = B`。

例：

`{1,2,3} = {3,2,1,1}`

因为集合无序，且元素不重复计算。

### 高考教练告诉你

高考喜欢把同一个集合换一种写法：

`{x | x²-5x+6=0}` 和 `{2,3}`

表面不同，解出来相同，所以集合相等。

### 知识联系

相等集合以后会继续出现在方程解集、不等式解集、函数定义域化简中。判断相等不要看外表写法，要回到“元素是否完全一样”。

## 知识块五：空集的重要性质

空集记作 `∅`。

重要性质：

1. `∅` 是任何集合的子集。
2. `∅` 是任何非空集合的真子集。

例：

- `∅ ⊆ {1,2,3}`
- `∅ ⊂ {1,2,3}`
- `∅ ⊂ ∅` 不成立，因为两个集合相等，不能是真子集。

### 最容易错的地方

- 把 `∅` 当作数字 0。
- 写所有子集时漏掉 `∅`。
- 认为 `∅ ⊂ ∅` 成立。

### 知识联系

空集以后会出现在方程无解、不等式无解、函数定义域为空、参数范围不存在等问题中。看到“没有任何元素”时，要主动想到空集，而不是写成 0 或无限集。

## 知识块六：区间表示

不等式集合可以用区间表示。

| 描述法 | 区间 |
|---|---|
| `{x | x>2}` | `(2,+∞)` |
| `{x | x≥2}` | `[2,+∞)` |
| `{x | -1<x≤3}` | `(-1,3]` |
| `{x | -1≤x<3}` | `[-1,3)` |

括号规则：

- 小括号：不包含端点。
- 中括号：包含端点。

### 知识联系

区间表示以后会贯穿函数定义域、单调区间、不等式解集、导数讨论区间。集合语言学清楚，后面函数语言才不会乱。

## 知识块七：数轴法

集合区间题统一采用数轴法。

### 方法总结

1. 画数轴。
2. 标端点。
3. 判断端点是否包含。
4. 画方向。
5. 比较范围包含关系。

例：

`A={x | x≥3}`，`B={x | x>1}`。

所有 `x≥3` 的数都满足 `x>1`，所以 `A ⊆ B`。  
又因为 `B` 中有 2 不属于 A，所以 `A ⊂ B`。

### 知识联系

数轴法以后会继续用于一元二次不等式、函数定义域、参数取值范围、集合交并补。区间题不要只凭感觉，统一画数轴。

## 高考风格题 1

设集合 `A={x | x²-5x+6=0}`，`B={2,3}`，则 A 与 B 的关系是：

A. `A ⊂ B`  
B. `B ⊂ A`  
C. `A = B`  
D. A 与 B 无包含关系

思路：先解方程，再比较集合。

## 高考风格题 2

设 `A={x | x≥3}`，`B={x | x>1}`，则 A 与 B 的关系是：

A. `A ⊂ B`  
B. `B ⊂ A`  
C. `A = B`  
D. A 与 B 无包含关系

思路：用数轴看范围包含。
