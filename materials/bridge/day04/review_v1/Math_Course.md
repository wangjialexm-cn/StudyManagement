# Day4 数学课程审核稿｜集合（二）

## Student Book

### 基本信息

- 学生：茄子
- 阶段：初升高衔接
- 主题：集合（二）
- 课程时长建议：60～80 分钟
- 本课定位：Day3 集合基本概念之后，进入集合关系与高考常见题型

### 1. 学习目标

本节课结束后，茄子应能：

1. 区分元素与集合、集合与集合之间的符号关系。
2. 理解子集、真子集、相等集合。
3. 掌握空集的重要性质。
4. 会用数轴法处理简单区间集合。
5. 能识别高考集合第一类常考题：集合关系、集合相等、区间包含。

### 2. 为什么学习这个知识点

Day3 学的是“集合里有什么”。Day4 学的是“集合和集合之间是什么关系”。

后面学习函数、不等式、定义域、值域、参数范围时，经常会遇到：

- 一个范围是否包含另一个范围；
- 两个解集是否相等；
- 某个参数取值范围是否满足题目要求。

这些问题本质上都是集合关系。

### 3. 高考教练告诉你

集合在高考中通常不是压轴题，但常出现在选择题前几题。它喜欢考：

- 符号是否混用：`∈`、`⊆`、`⊂`
- 子集和真子集是否分清
- 空集是否漏掉
- 区间端点是否看清
- 不等式集合能否用数轴快速判断

这类题不难，但容易因为概念不严谨丢分。

### 4. 知识块一：元素属于集合 vs 集合包含集合

连续讲授：

- `∈` 用在“元素”和“集合”之间。
- `⊆`、`⊂` 用在“集合”和“集合”之间。

例：

- `2 ∈ {1,2,3}`
- `{2} ⊆ {1,2,3}`

最容易错的地方：

- 写成 `2 ⊆ A`。
- 写成 `{2} ∈ A`，但 A 中并没有“集合 {2}”这个元素。

方法总结：

先判断符号两边是什么：

- 左边是单个数、字母、对象：考虑 `∈`
- 左边是 `{ }` 表示的整体：考虑 `⊆` 或 `⊂`

### 5. 知识块二：子集

定义：

如果集合 A 中的每一个元素都属于集合 B，就说 A 是 B 的子集，记作：

`A ⊆ B`

例：

`A={1,2}`，`B={1,2,3}`，则 `A ⊆ B`。

为什么学：

子集是判断“范围是否被包含”的基础。

高考教练告诉你：

判断子集不要靠感觉，要逐个检查 A 的元素是否都在 B 中。

最容易错的地方：

- 只看两个集合有没有共同元素。
- 忘记“每一个元素都要满足”。

### 6. 知识块三：真子集

定义：

如果 `A ⊆ B`，并且 `A ≠ B`，就说 A 是 B 的真子集，记作：

`A ⊂ B`

例：

`{1,2} ⊂ {1,2,3}`。

但：

`{1,2,3} ⊆ {1,2,3}` 成立，`{1,2,3} ⊂ {1,2,3}` 不成立。

方法总结：

判断真子集分两步：

1. 先判断是不是子集；
2. 再判断两个集合是否不相等。

最容易错的地方：

把“子集”直接当成“真子集”。

### 7. 知识块四：相等集合

两个集合元素完全相同，就相等。

也可以理解为：

如果 `A ⊆ B` 且 `B ⊆ A`，那么 `A = B`。

例：

`{1,2,3} = {3,2,1,1}`

因为集合不看顺序，也不重复计算元素。

高考教练告诉你：

高考中经常把一个集合写成描述法，一个集合写成列举法，让你判断是否相等。

例：

`{x | x² - 5x + 6 = 0} = {2,3}`

### 8. 知识块五：空集的重要性质

空集记作 `∅`。

重要性质：

1. `∅` 是任何集合的子集。
2. `∅` 是任何非空集合的真子集。

例：

- `∅ ⊆ {1,2,3}`
- `∅ ⊂ {1,2,3}`

为什么学：

写所有子集时，空集一定要写；讨论参数范围时，也可能出现空集情况。

最容易错的地方：

- 把 `∅` 当成数字 0。
- 漏写空集。
- 认为 `∅ ⊂ ∅` 成立，实际上不成立，因为真子集要求不相等。

### 9. 知识块六：区间表示与数轴法

当集合由不等式表示时，统一用数轴法。

常见区间：

- `{x | x > 2}` 可以理解为 `(2, +∞)`
- `{x | x ≥ 2}` 可以理解为 `[2, +∞)`
- `{x | -1 < x ≤ 3}` 可以理解为 `(-1, 3]`

方法总结：集合区间题统一采用数轴法。

步骤：

1. 画数轴；
2. 标端点；
3. 判断端点是否包含；
4. 看方向；
5. 比较两个范围谁包含谁。

高考教练告诉你：

区间题最容易错在端点。`>` 和 `≥` 差一个等号，答案可能完全不同。

### 10. 高考风格题

#### 高考风格题 1：集合相等

设集合 `A={x | x²-5x+6=0}`，`B={2,3}`，则 A 与 B 的关系是：

A. `A ⊂ B`  
B. `B ⊂ A`  
C. `A = B`  
D. `A ∩ B = ∅`

讲解思路：

先解 A：`x²-5x+6=0`，得 `x=2` 或 `x=3`，所以 `A={2,3}`，因此 `A=B`。

#### 高考风格题 2：区间包含

设 `A={x | x≥3}`，`B={x | x>1}`，则 A 与 B 的关系是：

A. `A ⊂ B`  
B. `B ⊂ A`  
C. `A = B`  
D. A 与 B 无包含关系

讲解思路：

用数轴法：所有 `x≥3` 的数都满足 `x>1`，但 `B` 中的 2 不属于 `A`。所以 `A ⊂ B`。

### 11. 本节知识地图

```text

├─ 空集性质：∅ 是任何集合的子集

```

## Class Practice

### A组｜符号区分

1. 设 `A={1,2,3}`，判断应填 `∈` 还是 `⊆`：  
   `2 ____ A`，`{2} ____ A`

2. 判断下列写法是否合理，并说明理由：  
   `2 ⊆ {1,2,3}`

### B组｜子集

3. 设 `A={1,2}`，`B={1,2,3}`。判断 `A ⊆ B` 是否成立。

4. 设 `C={2,4}`，`D={1,2,3}`。判断 `C ⊆ D` 是否成立。

### C组｜真子集与相等集合

5. 判断 `{1,2}` 是否为 `{1,2,3}` 的真子集。

6. 判断 `{1,2,3}` 是否为 `{3,2,1,1}` 的真子集。

7. 判断 `{x | x²-5x+6=0}` 与 `{2,3}` 是否相等。

### D组｜空集性质

8. 写出 `{1,2}` 的所有子集。

9. 判断 `∅ ⊆ {0}` 是否成立。

10. 判断 `∅ ⊂ ∅` 是否成立。

### E组｜数轴法

11. 设 `A={x | x>2}`，`B={x | x>0}`，判断 A 与 B 的关系。

12. 设 `A={x | x≥3}`，`B={x | x>3}`，判断 B 是否为 A 的真子集。

1. 设 `A={1,3}`，`B={1,2,3,4}`，判断 A 是否为 B 的子集。

10. 已知 `A={1,2}`，写出 A 的所有真子集。

## Teacher Guide

### 本节教学目标

- 让学生从“集合里有什么”升级到“集合之间有什么关系”。
- 区分 `∈`、`⊆`、`⊂`。
- 掌握子集、真子集、相等集合、空集性质。
- 初步形成集合区间题统一用数轴法的习惯。

### 建议教学流程

1. Day3 复测：四条腿的动物、小于0的正整数集合、描述法限制条件。
2. 连续讲授符号区分 + 子集，约 8 分钟。
3. 做 A/B 组课堂练习。
4. 连续讲授真子集 + 相等集合，约 10 分钟。
5. 做 C 组课堂练习。
6. 连续讲授空集性质，约 8 分钟。
7. 做 D 组课堂练习。
8. 连续讲授区间表示 + 数轴法，约 10 分钟。
9. 做 E 组课堂练习。
10. 讲高考风格题，最后布置 Homework。

### 推荐提问

- `2∈A` 和 `{2}⊆A` 有什么区别？
- 子集和真子集差在哪里？
- 为什么 `∅⊂∅` 不成立？
- 数轴法里，空心点和实心点分别表示什么？

### 学生可能出现的错误

- 混用 `∈` 和 `⊆`。
- 认为子集一定是真子集。
- 写子集时漏掉空集。
- 忘记真子集不允许相等。
- 区间端点包含关系判断错误。

### 如何根据学生回答调整讲解

- 若符号混乱：回到“元素 vs 集合”表格。
- 若真子集判断错：让学生找“另一个集合中多出来的元素”。
- 若空集性质不理解：先不抽象证明，用“没有反例”解释。
- 若数轴题错：先单独训练端点，再训练方向。
