# Day4 Change Log｜相对 Day3 的变化

> 本文件只记录 Day4 相比 Day3 的课程变化。

## 一、整体变化

√ 从“数学主课 + 三门副课学习单”升级为“四门正式课程”。  
√ 所有课程均包含 Student Book / Class Practice / Homework / Teacher Guide / Answer 的设计方向。  
√ Student Book 定位为课堂教材，不再是单纯自学教材。  
√ 课程节奏调整为“高中名师 + 家庭教师”模式。  
√ 增加 ChatGPT 教研审核流程，未审核前不生成最终 Word / PDF。  

---

## 二、数学变化

√ 增加子集。  
√ 增加真子集。  
√ 增加相等集合。  
√ 增加空集的重要性质。  
√ 增加区间表示。  
√ 增加数轴法。  
√ 增加高考集合第一类题型。  
√ 增加“高考教练告诉你”。  
√ 增加“方法总结”。  
√ 增加“最容易错的地方”。  
√ 增加“本节知识地图”。  
√ 增加 2 道高考风格题。  
√ 知识密度明显高于 Day3。

## 四、物理变化

√ 不再是平均速度学习单。

√ 增加“本节知识地图”。

## 五、化学变化

√ 不再是摩尔质量学习单。  
√ 升级为正式化学课程。  
√ 从摩尔质量复习推进到物质的量入口。

√ 增加“最容易错的地方”。  
√ 增加“本节知识地图”。  

---

## 六、流程变化

√ Codex 先开发审核包。  
√ 用户发送 Review Package 给 ChatGPT。  
√ ChatGPT 完成教研审核。  
√ Codex 根据审核意见修改。  
√ 审核通过后再生成最终版。  
√ 最终版再生成 Word / PDF / 打印版。  
√ 课堂完成后再批改并更新学习档案。
