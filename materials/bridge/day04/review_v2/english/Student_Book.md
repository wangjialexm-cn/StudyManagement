# Day4 英语 Student Book｜主谓一致与主动输出

## 课程定位

Day3 暴露的问题是：`Success depend on...` 应为 `Success depends on...`。这说明茄子不是不认识单词，而是在主动输出句子时语法控制还不稳定。

本课从“背单词”升级为“词汇 + 语法 + 长句 + 翻译 + 主动输出”。

建议课堂时长：35～45 分钟。

## 学习目标

1. 掌握 `depend on` 的正确句型。
2. 理解一般现在时中第三人称单数动词变化。
3. 能用 `increase`、`decrease`、`discover`、`especially` 写句子。
4. 能完成简单中译英，并检查主谓一致。

## 为什么学习这个知识点

高中英语不是只背 3500 词。真正影响分数的是：能否把词放进正确句子。

`depend` 认识不难，但 `Success depends on practice.` 写错，就会暴露主动输出能力不足。

## 高考教练告诉你

高考英语常见扣分点：

- 语法填空中动词形式错误；
- 写作中主谓不一致；
- 认识单词但写不出句子；
- 中译英时只翻关键词，句子结构不完整。

所以本课重点是输出准确，而不是多背几个词。

## 知识块一：depend on

`depend on` 表示“取决于，依靠”。

例句：

- `Success depends on hard work.`
- `The result depends on practice.`
- `Your progress depends on daily effort.`

### 方法总结

写 `depend on` 句子时：

1. 先找主语；
2. 判断主语是否单数；
3. 单数主语后写 `depends on`。

### 最容易错的地方

`Success` 是单数概念，所以不能写 `Success depend on...`。

## 知识块二：increase / decrease

- `increase`：增加
- `decrease`：减少，下降

例句：

- `The number increases.`
- `The temperature decreases.`
- `Practice increases confidence.`

### 方法总结

看到“增加/下降”，先判断主语是谁，再决定动词形式。

## 知识块三：discover / especially

- `discover`：发现
- `especially`：尤其，特别

例句：

- `I discovered a useful method.`
- `I especially like this example.`

### 最容易错的地方

`discover` 是动词，`discovery` 是名词，不要混用。

## 知识块四：中译英输出

中译英不是逐词翻译，而是先搭句子结构。

例：

“成功取决于每天练习。”

先找主语：成功 → `Success`  
再找谓语：取决于 → `depends on`  
再补充内容：每天练习 → `daily practice`

完整句：

`Success depends on daily practice.`

## Mini Reading：词汇、语法、阅读合在一起

Read the passage and underline the five key words.

```text
Many students discover that their progress does not depend on luck.
It depends on daily practice, especially when they learn a new language.
If they read and write every day, their confidence will increase.
If they only remember words but never use them, their accuracy may decrease.
A good learner checks the subject first and then chooses the correct verb form.
This small habit helps students write clearer English sentences.
```

课堂任务：

1. 找出 `depend`、`increase`、`decrease`、`discover`、`especially`。
2. 翻译前两句。
3. 找出文中两个含有第三人称单数的句子。
4. 仿写一句：`My progress depends on ________.`

### 方法总结

阅读不是只看懂大意。本课阅读要按顺序完成：

`讲解 → 阅读 → 找重点表达 → 翻译 → 主动输出`

## 本节知识地图

```text
主动输出
├─ 词汇：depend / increase / decrease / discover / especially
├─ 语法：第三人称单数
├─ 句型：Success depends on...
├─ 翻译：先搭结构，再填词
└─ 输出检查：主语 → 动词 → 搭配
```

## 本节一句话总结

先找主语，再决定动词形式。
