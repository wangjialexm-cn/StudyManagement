# 英语学情诊断与假期补漏清单

整理日期：2026-06-23  
资料来源：原“英语复习”项目中的错题归纳、单词掌握表、句型掌握表及第 33—37 轮复习记录。

## 一、目前英语学习情况概览

- 已积累约 214 个单词和短语的复习记录，并完成至少 37 轮学习或复习。
- 看英文辨认中文的表现明显好于“看中文主动拼出英文”。第 33—35 轮主动拼写首答正确率约为 42%。
- 近期较稳定的词包括：`remain`、`receive`、`record`、`remind`；`reduce` 已有进步，但历史上反复遗忘，仍需抽查。
- 当前最值得优先处理的不是继续大量增加词汇，而是：拼写、易混词、时态、被动语态、词形变化和交际句型。

## 二、词汇分层整理

### A 级：反复出错，优先回炉

| 单词或短语 | 核心意思 | 已出现的问题 |
|---|---|---|
| reduce | 减少；降低 | 多轮先答不出，近期才开始直接答对 |
| remain | 保持；仍然；剩余 | 曾高频遗忘，近期表现改善 |
| properly | 正确地；恰当地 | 反复与 `probably` 混淆 |
| purpose | 目的；用途 | 多次遗忘，曾误联想到“当然” |
| image | 图像；形象 | 主动回忆困难，释义不完整 |
| realize | 意识到；认识到 | 容易只想到“实现”或与 `real` 混淆 |
| replace | 替换；取代 | 隔轮仍不会，需与 `remove` 区分 |
| compare | 比较；对比 | 隔轮仍不会 |
| refuse | 拒绝 | 隔轮仍不会 |
| mental | 心理的；精神的 | 多次遗忘 |
| shock | 震惊；使震惊 | 曾与 `shoot` 混淆 |
| condition | 条件；状况 | 曾答成 `confidence` |
| amusement | 娱乐 | 拼写多次不稳 |
| available | 可用的；有空的 | 拼写不稳，易与“可行的”混淆 |
| research | 研究 | 曾拼成 `reserch` |
| destroy | 摧毁；破坏 | 需要提示才能写出 |
| defeat | 打败 | 曾拼成 `defect` |
| performance | 表演；表现 | 曾答成 `performer` |
| preparations | 准备工作 | 曾写成 `preparings` |
| excitement | 激动；兴奋 | 曾拼成 `excitedment` |

### B 级：意思基本知道，主动拼写不稳

`achieve`、`government`、`include`、`rapid`、`pacific`、`support`、`purpose`、`probably`、`science`、`perfect`、`available`、`modern`、`research`、`excitement`、`preparations`。

典型拼写错误：

- `achieve` → `acheive`
- `government` → `gavermant`
- `include` → `incloud`
- `probably` → `probobly`
- `support` → `sppourt`
- `purpose` → `porpuse`
- `science` → `seience`
- `modern` → `morden`

### C 级：近期表现较稳定，低频抽查即可

`receive`、`record`、`remind`、`protect`、`spirit`、`impossible`、`whales`、`national`、`performance`、`support`、`capital`、`slim`、`advantage`、`pardon`。

“相对稳定”不等于永久掌握，建议每隔一至两周抽查一次中文到英文的主动拼写。

## 四、语法薄弱点

### 1. 时态

- 一般过去时：`yesterday`、`long ago` 后动词要用过去式。
- 过去进行时：`at that moment`、`at nine yesterday` 表示过去某时正在进行。
- 现在完成时：`How long have you lived...?`；不能写成 `have been became`。
- 主将从现：`If it snows, they will make a snowman.`

### 2. 被动语态

结构：`be + 过去分词`。

重点错例：

- `The documentary was shown in Italy.`
- `Nanyin is played with traditional instruments.`
- `The airport will be completed by the end of this year.`
- `The building was washed away.`

### 3. 情态动词

- `mustn't`：禁止。
- `needn't / don't have to`：不必。
- `can't`：不可能。
- `Must I...?` 的否定回答：`No, you needn't.`
- 情态动词后接动词原形。

### 4. 非谓语和固定结构

- `keep / enjoy / feel like / mind + doing`
- `be interested in doing`
- `spend money doing`
- `think of doing`
- `by doing`
- `Thank you for doing`

### 5. 比较级与最高级

- `much + 比较级`：`much better`、`much thinner`
- `of all` 常用最高级：`most beautifully`
- 多音节最高级：`the most famous`
- `as + 原级 + as`

### 6. 词形与代词

- 形容词与副词：`traditional → traditionally`、`deep → deeply`、`confident → confidently`
- `it`、`its`、`it's`、`itself` 容易混淆。
- `them` 与 `themselves`、`he` 与 `himself` 要根据句子成分选择。

## 五、交际与句型薄弱点

### 优先背熟的 12 个句型

1. `What does your father do?`
2. `What are you going to do tomorrow?`
3. `What's the weather like today?`
4. `How long have you lived in this city?`
5. `How many apples are there on the table?`
6. `Thank you for inviting me.`
7. `How's it going?`
8. `Would you like to go to the movies with me this evening?`
9. `How about going to the park?`
10. `Would you mind opening the window?`
11. `You'd better not go out alone at night.`
12. `I'm sorry to hear that.`

### 仍容易失分的交际规则

- 接受邀请：`I'd love to.`，不能漏掉 `to`。
- 同意 `Would you mind...?`：`Of course not. / Not at all.`
- 回应道歉：`It doesn't matter. / That's all right.`
- 对坏消息表示遗憾：`I'm sorry to hear that.`
- 电话中自我介绍：`This is ... speaking.`
- 祝贺别人：`Congratulations!`
- 问日期：`What's the date today?`
- 问星期：`What day is it today?`

## 六、写作与综合运用情况

- 两篇活动类作文曾未作答，说明完整写作训练不足。
- 看图写句的主要问题不是没有思路，而是时态、主语、冠词和固定搭配不稳。
- 情景交际中容易出现“意思差不多，但不符合标准表达”的情况。
- 阅读和完形中的薄弱点主要是语境选词、人物情绪变化和短语整体意义，不能逐词翻译。

建议假期先练活动类作文：开头交代时间、地点和目的；中间按顺序写 2—3 项活动；结尾写感受和意义。

## 七、假期第一阶段的建议顺序

### 第 1 周：稳定初中基础

- 每天复习 8 个 A 级词汇，采用“中文写英文”。
- 每天练 2 个核心句型，并进行中文到英文默写。
- 隔天完成一组时态或被动语态练习。
- 周末写一篇 80—100 词的活动类作文。

### 第 2 周：易混词与综合运用

- 每天复习 5 组易混词，每个词各造一个短句。
- 专练情态动词、非谓语、比较等级和词形变化。
- 完成两组短文填词及两组情景交际。
- 修改并重写第 1 周作文。

### 第 3 周：检测后衔接高中

- 用一套初中综合卷复测；只把仍然出错的知识点带入下一轮。
- 开始高一词汇、句子成分和长难句入门。
- 初中补漏与高中预习时间约按 1:1 分配。

## 八、暂不急于做的事

- 暂不大量背新词，先把高频错词和主动拼写补稳。
- 暂不追求偏难语法题，先确保基础结构能写对。
- 暂不把“提示后立刻答对”记为掌握；至少隔一轮还能主动写出，才算真正巩固。
