# 学科能力档案

## 使用说明

- 本文件记录当前能力状态，不罗列全部学习过程。
- 评价等级：`信息不足`、`薄弱`、`掌握一般`、`基本掌握`、`稳定掌握`。
- 每项判断必须写明证据来源和最近验证日期。
- 能力变化后保留简短更新说明，详细过程写入 `learning_record.md`。

- 从 v2.0 起，长期追踪逐步采用 L1~L10 能力等级。
