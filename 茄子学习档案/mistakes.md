## 收录规则

- 独立作答错误、空题、表达不规范或依赖提示才完成的题目均可收录。
- 同一知识点反复出错时，保留代表题并累计次数，避免无意义抄题。
- 错题必须记录错因、正确方法、订正和间隔复测结果。
- 只有间隔复测通过，错题状态才可标记为“已巩固”。

## 状态说明

- `待订正`：尚未完成正确订正。
- `待复测`：已订正，但尚未间隔验证。
- `复测未通过`：再次独立作答仍有错误。
- `已巩固`：间隔复测独立完成正确。

### M-0001｜2026-06-26｜数学｜集合确定性

- 来源：Day3 数学集合第一课课堂练习
- 教研复核：已纳入 Day3 教研纪要最终版
- 原题：判断“四条腿的动物”是否能构成集合。
- 学生原答案：误认为不是集合。
- 正确答案：可以构成集合。
- 错误类型：概念混淆
- 错因分析：容易把“数量很多”误认为“标准不明确”。实际上“四条腿”是可判断标准，对象数量多不影响构成集合。
- 正确方法：判断能否构成集合时，先看标准是否明确，而不是看对象多不多。
- 一题一题讲解摘要：给出任意动物，可以判断它是否“四条腿”，因此标准明确。
- 同类变式：判断“会飞的动物”“比较可爱的动物”是否能构成集合。

- 首次订正日期：信息不足

### M-0002｜2026-06-26｜数学｜空集

- 来源：Day3 数学集合第一课课堂练习
- 教研复核：已纳入 Day3 教研纪要最终版
- 原题：判断“小于 0 的正整数集合”是什么集合。
- 学生原答案：写成无限集。
- 正确答案：空集。
- 错误类型：概念混淆
- 错因分析：看到“整数”容易联想到无限多个，但忽略了“小于 0”和“正整数”两个条件不能同时满足。
- 正确方法：先逐条检查限制条件，若没有任何元素满足，则为空集。
- 一题一题讲解摘要：正整数为 1,2,3,...，没有小于 0 的正整数。
- 同类变式：`{x | x 是大于 0 的负整数}`。

### M-0003｜2026-06-26｜数学｜描述法限制条件

- 来源：Day3 数学集合第一课课堂练习
- 教研复核：已纳入 Day3 教研纪要最终版
- 原题：描述法表达相关题。
- 学生原答案：容易遗漏限制条件。
- 正确答案：应完整写出对象范围和限制条件。
- 错误类型：表达不规范
- 错因分析：能理解描述法，但表达时容易少写“不大于”“正整数”“倍数”等限制。
- 正确方法：描述法按“对象 + 全部条件”检查，写完后逐条回看条件是否遗漏。
- 一题一题讲解摘要：`{x | x 是不大于 12 的正整数，且 x 是 3 的倍数}` 中“不大于 12”“正整数”“3 的倍数”都不能省。
- 同类变式：用描述法表示 `{2,4,6,8}`。

### M-0004｜2026-06-26｜英语｜Success depends on...

- 来源：Day3 英语副课学习单
- 原题：表达“Success depends on...”
- 学生原答案：`Success depend on...`
- 正确答案：`Success depends on...`
- 错误类型：表达不规范
- 错因分析：主动输出时遗漏第三人称单数动词变化。
- 正确方法：主语 `Success` 为单数概念，谓语动词应使用 `depends`。
- 一题一题讲解摘要：单数主语 + 一般现在时动词三单形式。
- 同类变式：`This plan depends on time.` / `The result depends on practice.`

### M-0005｜2026-06-26｜化学｜CO₂ 摩尔质量

- 来源：Day3 化学副课：摩尔质量复习
- 原题：计算 CO₂ 的摩尔质量。
- 学生原答案：40 g/mol
- 正确答案：44 g/mol
- 错误类型：计算失误
- 错因分析：计算 CO₂ 时未正确计算两个氧原子的质量贡献。
- 正确方法：CO₂ = C + 2O = 12 + 2×16 = 44 g/mol。
- 一题一题讲解摘要：看清下标，先乘后加。
- 同类变式：H₂O、O₂、CaCO₃ 的摩尔质量。
