# 错题库

## 收录规则

- 独立作答错误、空题、表达不规范或依赖提示才完成的题目均可收录。
- 同一知识点反复出错时，保留代表题并累计次数，避免无意义抄题。
- 错题必须记录错因、正确方法、订正和间隔复测结果。
- 只有间隔复测通过，错题状态才可标记为“已巩固”。
- 从 2026-06-27 起，本错题库执行 C+模式（错题变式训练系统）。

## C+模式错题系统规则

### 1. 错题必须分类

错题按学科分类：

- 数学
- 物理
- 化学
- 英语

### 2. 错题必须标注

每个错题必须标注：

- 错误类型
- 知识点标签
- 掌握等级（1～5 星）

### 3. 变式训练系统

每个错题必须生成：

- 基础变式题：1 题
- 提升变式题：1 题
- 综合变式题：1 题

### 4. 错题循环复现规则

- 错一次：第二天出现。
- 再错：Day+2 再出现。
- 连续两次正确：移出错题库或标记为“已巩固”。

### 5. 掌握等级说明

| 等级 | 含义 | 处理方式 |
|---|---|---|
| 1★ | 基本不会 | 重新讲解，立即安排基础变式 |
| 2★ | 有明显漏洞 | 第二天复现，并安排基础 + 提升变式 |
| 3★ | 订正后能理解，但表达或迁移不稳 | 安排提升变式和复现 |
| 4★ | 已有一次独立正确记录 | 等待第二次连续正确 |
| 5★ | 连续两次独立正确 | 标记已巩固，可移出活跃错题循环 |

> 星级只表示当前错题掌握状态，不等同于整门学科水平。

## 状态说明

- `待订正`：尚未完成正确订正。
- `待复测`：已订正，但尚未间隔验证。
- `复测未通过`：再次独立作答仍有错误。
- `已巩固`：间隔复测独立完成正确。

### M-0001｜2026-06-26｜数学｜集合确定性

- 来源：Day3 数学集合第一课课堂练习
- 教研复核：已纳入 Day3 教研纪要最终版
- 原题：判断“四条腿的动物”是否能构成集合。
- 学生原答案：误认为不是集合。
- 正确答案：可以构成集合。
- 错误类型：概念混淆
- 知识点标签：集合确定性
- Root Cause：集合判断时缺少“先证明标准是否明确”的意识，容易把对象数量多误判为标准不明确。
- 错因分析：容易把“数量很多”误认为“标准不明确”。实际上“四条腿”是可判断标准，对象数量多不影响构成集合。
- 正确方法：判断能否构成集合时，先看标准是否明确，而不是看对象多不多。
- 一题一题讲解摘要：给出任意动物，可以判断它是否“四条腿”，因此标准明确。
- 掌握等级：2★
- C+变式训练：
  - 基础变式题：判断“会飞的动物”能否构成集合，并说明理由。
  - 提升变式题：判断“跑得很快的动物”能否构成集合，并说明理由。
  - 综合变式题：在“会飞的动物”“比较可爱的动物”“四条腿的动物”中，选出能构成集合的对象，并说明判断标准。

- 来源：Day3 数学集合第一课课堂练习

- 原题：判断“小于 0 的正整数集合”是什么集合。
- 学生原答案：写成无限集。
- 正确答案：空集。
- 错误类型：概念混淆
- 知识点标签：空集
- Root Cause：条件检索不完整，没有逐条验证“小于0”和“正整数”是否能同时成立。
- 错因分析：看到“整数”容易联想到无限多个，但忽略了“小于 0”和“正整数”两个条件不能同时满足。
- 正确方法：先逐条检查限制条件，若没有任何元素满足，则为空集。
- 一题一题讲解摘要：正整数为 1,2,3,...，没有小于 0 的正整数。
- 掌握等级：2★
- C+变式训练：

  - 提升变式题：写出“大于 0 的负整数集合”。
  - 综合变式题：判断 `{x | x 是正整数，且 x<0}` 与 `∅` 是否相等，并说明理由。

### M-0003｜2026-06-26｜数学｜描述法限制条件

- 原题：描述法表达相关题。
- 学生原答案：容易遗漏限制条件。
- 正确答案：应完整写出对象范围和限制条件。
- 错误类型：表达不规范
- 知识点标签：描述法限制条件
- Root Cause：数学表达完整性不足，写描述法时没有建立“对象 + 全部限制条件”的检查习惯。
- 错因分析：能理解描述法，但表达时容易少写“不大于”“正整数”“倍数”等限制。
- 正确方法：描述法按“对象 + 全部条件”检查，写完后逐条回看条件是否遗漏。
- 一题一题讲解摘要：`{x | x 是不大于 12 的正整数，且 x 是 3 的倍数}` 中“不大于 12”“正整数”“3 的倍数”都不能省。

  - 提升变式题：用描述法表示 `{3,6,9,12}`，要求写出对象范围和限制条件。
  - 综合变式题：判断 `{x | x 是 12 以内的 3 的倍数}` 是否完整表示 `{3,6,9,12}`，若不完整请补充限制条件。

- 错误类型：计算失误

  - 综合变式题：比较 CO₂ 和 CaCO₃ 的摩尔质量计算过程，说明下标在计算中的作用。

### M-0006｜2026-06-27｜物理｜平均速度定义表达

- 来源：Day4 物理正式课程批改
- 原题：平均速度定义或概念表达题。
- 学生原答案：平均速度定义表达不规范。
- 正确答案：平均速度描述整个运动过程，应根据总路程和总时间计算，即 `v=s/t`。
- 错误类型：表达不规范
- 知识点标签：平均速度定义表达
- 掌握等级：3★
- Root Cause：概念表达能力不足；会用公式 `v=s/t`，但不能稳定用语言解释“总路程 / 总时间”的物理意义。
- 错因分析：会使用公式，但用语言说明物理意义时不够规范；需要从“会算”升级到“会解释”。
- 正确方法：表达平均速度时必须包含“整个过程”“总路程”“总时间”三个关键词。
- 一题一题讲解摘要：平均速度不是某一瞬间速度，也不是简单把两个速度相加除以 2，而是全程总路程除以总时间。
- C+变式训练：

  - 综合变式题：设计一个分段运动例子，写出总路程、总时间、平均速度，并用文字解释计算依据。

### M-0007｜2026-06-27｜英语｜现在进行时 be + doing

- 来源：Day4 英语正式课程批改
- 原题：`be + doing` 相关输出题。
- 学生原答案：`be + doing` 结构错误。
- 正确答案：现在进行时应使用 `主语 + be动词 + doing`。
- 错误类型：语法结构错误
- 知识点标签：现在进行时 be + doing
- 掌握等级：2★
- Root Cause：Day5 复盘后修正为词汇主动输出不足：`increase / decrease` 没有进入主动输出阶段，不知道其动词身份；不是不会 `be doing`。
- 错因分析：Day4 表面上出现 `be + doing` 结构错误；Day5 已能正确写出 `is increasing`、`is decreasing`，因此根因修正为词汇身份识别不足。

- 一题一题讲解摘要：例如“他正在读书”应写成 `He is reading.`，不能漏 `is`，也不能写成原形动词。
- C+变式训练：

  - 提升变式题：中译英：她正在写作业。
  - 综合变式题：用现在进行时写 3 句话，分别以 `I`、`She`、`They` 作主语。
